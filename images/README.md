# Every Witch Way - Image Guide

Download images from Unsplash and save them with the following filenames:

## Home Page (index.html)
- `hero-bg.jpg` - Cozy cafe atmosphere, warm lighting, autumn vibes (1920x1080 recommended)
- `featured-latte.jpg` - Pumpkin spice latte or seasonal coffee drink (600x400)
- `featured-croissants.jpg` - Fresh baked croissants or pastries (600x400)
- `featured-mocha.jpg` - Mocha or artisan coffee drink (600x400)
- `cafe-interior.jpg` - Cozy cafe interior with warm autumn decor (800x600)

## Menu Page (menu.html)
- `menu-latte.jpg` - Signature latte with latte art (600x400)
- `menu-cake.jpg` - Chocolate lava cake or rich dessert (600x400)

## About Page (about.html)
- `founders.jpg` - Could be cafe owners or baristas at work (800x600)
- `team-elena.jpg` - Female baker/chef portrait (400x400, square)
- `team-marcus.jpg` - Male barista/coffee expert portrait (400x400, square)
- `team-sophie.jpg` - Female pastry chef portrait (400x400, square)

## Locations Page (locations.html)
- `location-downtown.jpg` - Historic building, outdoor seating cafe (800x500)
- `location-drphillips.jpg` - Modern cafe interior (800x500)
- `location-winterpark.jpg` - Charming storefront or garden courtyard (800x500)

- "coffee shop interior autumn"

## Image Optimization Tips
- Use JPG format for photographs
- Compress images for web (tools: TinyPNG, Squoosh)
- Keep file sizes under 200KB when possible
